# Searching Algorithms in Java

This repository contains Java implementations of **Linear Search** and **Binary Search**. The project is designed to strengthen the understanding of searching algorithms, their working principles, and their time and space complexities. These implementations are useful for learning Data Structures & Algorithms (DSA) and preparing for coding interviews and placement assessments.

---

## Algorithms Implemented

- Linear Search
- Binary Search

---

## Objectives

- Understand the fundamentals of searching algorithms.
- Learn the differences between Linear Search and Binary Search.
- Analyze the efficiency of searching algorithms.
- Build a strong foundation in Data Structures and Algorithms.
- Prepare for coding interviews and technical assessments.

## How to Run

### Clone the repository

```bash
git clone https://github.com/PalagiriNagayogeswari/Searching.git
```

### Navigate to the project directory

```bash
cd Searching
```

### Compile a Java file

```bash
javac FileName.java
```

### Run the program

```bash
java FileName
```

Replace `FileName` with the name of the Java file you want to execute.

## Time Complexity Comparison

| Algorithm | Best Case | Average Case | Worst Case | Space Complexity | Requirement |
|-----------|-----------|--------------|------------|------------------|-------------|
| Linear Search | O(1) | O(n) | O(n) | O(1) | Works on both sorted and unsorted data |
| Binary Search | O(1) | O(log n) | O(log n) | O(1) | Requires sorted data |
